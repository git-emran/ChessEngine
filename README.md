# Chess Engine

This is a chess engine written in python.

I will also guide you through the process of creating a chess engine.

#### How to run

Clone or download the project, run `python run ChessEngine.py` to open the game locally. Keep in mind This game uses `pygame`

### Architecture

Currently the The core logic and game piece logic has been seperated into `ChessEngine.py` and `ChessMain.py`. Chess Engine has all the pieces and their moces.
